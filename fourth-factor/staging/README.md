# Fourth Factor — Staging Layer

This folder is for ideas that arrived sideways, on low bandwidth, or mid-session without full review.

Nothing here is canon. Nothing here is locked. Nothing here feeds into architecture.md, plan.md, or the formal spec until explicitly promoted by Jay at full cognition.

## Protocol

- ECHO files unreviewed ideas here during sessions
- Jay reviews when bandwidth allows
- Promotion = copy to the right permanent file + delete from staging
- Abandon = delete without ceremony
- When in doubt: stage it, don't lock it

## Status flags used in this folder

`[UNREVIEWED]` — came in sideways, needs full-cognition eval  
`[PROMISING]` — Jay gave a thumbs up but didn't fully evaluate  
`[READY TO PROMOTE]` — Jay reviewed, wants it in the real files  
`[ABANDON]` — good idea for someone else's project, not this one
